# Connectify - Newcomer Community Platform

A full-stack mobile application built with Flutter (frontend) and Node.js (backend) to help newcomers connect with local communities, find jobs, access courses, and discover startup opportunities.

- **Home Feed**: Community posts and updates
- **Job Listings**: Browse and apply for jobs
- **Courses**: Access learning materials
- **Startup Discovery**: Find and connect with local startups
- **User Profiles**: Personal profiles with achievements
- **Multi-platform**: iOS, Android, Web, Desktop support

### Backend (Node.js/Express)
- **RESTful API**: Clean API structure
- **JWT Authentication**: Secure token-based auth
- **MongoDB Database**: NoSQL database for scalability
- **User Management**: CRUD operations for users
